# 🚗 Car Dealership Project 🏢

This is a full-stack web application for managing a car dealership. It provides features for user authentication, viewing available cars, managing dealership inventory, and more.

## Features ✨

- **Authentication**: Admin, user, and dealership authentication using JWT.
- **View Cars**: View all cars and filter by dealership.
- **Add Cars**: Dealerships can add cars to their inventory.
- **Manage Deals**: Add deals and view deals for a specific dealership or car.
- **User Dashboard**: Users can view dealerships with a certain car, view owned vehicles, and view deals on a certain car.
- **Dealer Dashboard**: Dealerships can manage their inventory, view sold vehicles, and add deals.

## Technologies Used 🛠️

- **Backend**: Node.js, Express.js, MongoDB
- **Frontend**: SvelteKit, Tailwind CSS
- **Other Tools**: JWT for authentication, Axios for HTTP requests
